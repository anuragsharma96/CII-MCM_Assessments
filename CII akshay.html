Q1. what is computer and how does it work?

Ans. Computer is an electroinc device and A computer is an electronic machine that processes information. It takes in raw data, stores it, processes it, and produces output. Computers operate through hardware and software, accepting input and providing output based on complex calculations..



Q2. Components of computer?

Ans. A computer is an electronic device that processes data, performs operations,
and displays results. It consists of various hardware and software components that work together to provide functionality. The primary components of
a computer include the Input Unit, Central Processing
Unit (CPU), and Output Unit

Q3. Peripheral devices and drivers?

Ans. A device peripheral is a hardware component that can be
connected to a computing device (such as a PC) to enhance its functionality
or performance. A device driver is a software program that acts as
a translator between the device peripheral and the operating system
of the device. The device driver communicates with the device peripheral
through the computer bus, which is used to connect the device
with the computer
